MIS PROYECTOS

Me complace presentar una visión general de algunos de los proyectos destacados que he llevado a cabo en el ámbito de la programación de páginas web. A lo largo de mi trayectoria, he tenido la oportunidad de aplicar mis habilidades y conocimientos en el desarrollo web para crear soluciones interactivas y visualmente atractivas. A continuación, detallo algunos de los proyectos más relevantes en los que he participado.

VLSYSTEM

Este proyecto fue desarrollado para una empresa de transporte de materiales y prestacion de servicios, como personal y maquinaria pesada. Tiene lenguajes tales como: Html, CSS, Javascript y PHP.

FRAGANTINO

Este fue mi segundo proyecto, el cual fue para una tienda de perfumes, la cual contiene pasarela de pago, diferentes entornos para catalogos, una pagina principal, carro de compras, entre otros. Tiene lenguajes de desarrollo como: Html, CSS, JavaScript, PHP. Tambien contiene elementos externos como lo es PSE.

GALAXY LUPPYX

Este es mi proyecto mas reciente, el cual fue un video juego. Se llama GALAXY-LUPPYX y es de tap tap, se desarrollo para un cliente que pedia algo entretenido pero no muy complejo, se desarrollo con Android Studio. Los lenguajes que se llevaron a cabo fueron: JAVA y PHP.
